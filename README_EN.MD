# About Shincolle 1.12.2β

This is an unofficial port mod from Shincolle 1.10.2. It includes some modifications from the original mod.<br><br>

## List of Changes

・The vortex of the "Hadal Hydrothermal Vortex" now rotates more smoothly.<br>
・Shinkaiseikan, ship girls will now actively follow the Admiral.<br>
・Shinkaiseikan, ship girls can now open trapdoors.<br>
・Shinkaiseikan, ship girls will now try their best to navigate using ladders.<br>
・Fixes have been applied to the inventory textures of Shinkaiseikan, ship girls.<br>
・Resolved an issue where right-clicking Shinkaiseikan, ship girls would cause the Admiral to use the item held in their hand.<br>
・Added a new configuration option, "Consumption Mode". Depending on the setting, the amount of fuel consumed for construction and the number of Shincolle-related items appearing in chests will change.<br>
・Placing the "Abyssal Admiral's Desk" side-by-side will now change its texture.<br>
・Added a new function to the "Abyssal Admiral's Desk". Selected Shinkaiseikan, ship girls will gather in front of the desk.<br>
・Fixed a major bug related to the "Abyssal Admiral's Desk".<br>
・Added a new block, the "Abyssal Admiral's Chair". \*The author is not a designer.<br>
・Added a new block, "Nether Abyssium Ore". It generates in the Nether. \*The author is not a designer.<br>
・Particles are now displayed on "Nether Abyssium Ore" and "Polymetallic Ore".<br>
・Changed the specifications of PVP mode. When the setting is off, Shinkaiseikan, ship girls will no longer counterattack if an Admiral attacks another Admiral's Shinkaiseikan, ship girls. (Probably)<br>
・Optimized internal programming, compressed images, and removed some sounds. As a result, the file size has been reduced by about 1MB.<br>
・Added new equipment to "Akatsuki".<br>
・Added a Japanese translation to the "Abyssal Logbook".<br>
・Characters displayed in the index of the "Abyssal Logbook" can now be moved more smoothly.<br>
・Selecting Shinkaiseikan, ship girls using the "Wo-Class Command Scepter" is now much faster.<br>
・Changed the logic for maritime formation movement.<br>
・Admirals can now eat the "Model 21 Zero Fighter (1/48)".<br>
・Admirals can now drink the "Repair Bucket".<br>
・Slightly improved compatibility with the Morph MOD.<br><br>
