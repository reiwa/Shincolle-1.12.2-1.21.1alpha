# 關於Shincolle 1.12.2β

這是從Shincolle 1.10.2非官方移植的MOD。包含對原MOD的部分修改。<br><br>

## 變更點一覽

・「超深海熱漩」的漩渦現在能旋轉得更加順暢了。<br>
・深海棲艦, 艦娘現在會更積極地跟隨提督了。<br>
・深海棲艦, 艦娘現在可以打開活板門（陷阱門）了。<br>
・深海棲艦, 艦娘現在會努力嘗試使用梯子移動了。<br>
・修正了深海棲艦, 艦娘的物品欄材質。<br>
・修復了對深海棲艦, 艦娘點擊右鍵時，提督會使用手中物品的問題。<br>
・新增了設定選項「Consumption Mode」。根據此設定，建造時消耗的燃料量以及寶箱中出現的Shincolle相關物品數量將會改變。<br>
・將「深海提督的辦公桌」橫向並排擺放時，材質現在會發生變化。<br>
・「深海提督的辦公桌」新增了功能。被選中的深海棲艦, 艦娘會在辦公桌前集結。<br>
・修復了關於「深海提督的辦公桌」的重大錯誤（Bug）。<br>
・新增了方塊「深海提督的椅子」。※作者並不是設計師。<br>
・新增了方塊「下界深海多金屬礦」。它會在下界生成。※作者並不是設計師。<br>
・「下界深海多金屬礦」和「多金屬礦」現在會顯示粒子效果。<br>
・更改了PVP模式的機制。當設定為關閉時，即使提督攻擊其他提督的深海棲艦, 艦娘，她們也不再進行反擊。（應該啦）<br>
・進行了內部程式最佳化、圖片壓縮並刪除了一些音效。因此，檔案大小減少了約1MB。<br>
・為「曉」新增了裝備。<br>
・為「深海日誌」新增了日文翻譯。<br>
・「深海日誌」圖鑑中顯示的角色現在可以更順暢地移動了。<br>
・使用「ヲ級的指揮權杖」選擇深海棲艦, 艦娘現在變得更迅速了。<br>
・更改了海上的陣型移動邏輯。<br>
・提督現在可以吃下「零式21型艦戰 (1/48)」了。<br>
・提督現在可以喝下「修復桶」了。<br>
・稍微提升了與Morph MOD的相容性。<br><br>

# 關於Shincolle 1.21.1α（neoforge）

這是從Shincolle 1.10.2非官方移植的MOD。包含大量未實裝的功能和Bug。<br>
・目前是創造模式專用的MOD。<br>
・深海棲艦, 艦娘會跟隨提督，並可以根據「ヲ級的指揮權杖」的指示進行移動和攻擊。<br>
・除此之外的其他功能目前均未實裝。<br>
