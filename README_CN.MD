# 关于Shincolle 1.12.2β

这是从Shincolle 1.10.2非官方移植的MOD。包含对原MOD的部分修改。<br><br>

## 变更点一览

・“超深海热漩”的漩涡现在会旋转得更加平滑。<br>
・深海栖舰, 舰娘现在会更积极地跟随提督了。<br>
・深海栖舰, 舰娘现在可以打开活板门了。<br>
・深海栖舰, 舰娘现在会努力尝试使用梯子移动了。<br>
・修正了深海栖舰, 舰娘的物品栏纹理。<br>
・修复了右键点击深海栖舰, 舰娘时，提督会使用手中物品的问题。<br>
・添加了新的设置项“Consumption Mode”。根据该设置，建造时消耗的燃料量以及箱子中出现的Shincolle相关物品的数量会发生变化。<br>
・将“深海提督的办公桌”横向并排摆放时，纹理现在会发生变化。<br>
・“深海提督的办公桌”添加了新功能。被选中的深海栖舰, 舰娘会在办公桌前集结。<br>
・修复了关于“深海提督的办公桌”的一个重大Bug。<br>
・添加了新方块“深海提督的椅子”。※作者并不是设计师。<br>
・添加了新方块“下界深海金属矿”。它会在下界生成。※作者并不是设计师。<br>
・“下界深海金属矿”和“多金属矿”现在会显示粒子效果。<br>
・更改了PVP模式的机制。当设置关闭时，即使提督攻击其他提督的深海栖舰, 舰娘，它们也不再进行反击。（大概吧）<br>
・进行了内部程序优化、图像压缩并删除了一些音效。因此，文件大小减少了约1MB。<br>
・为“晓”添加了新装备。<br>
・为“深海日志”添加了日语翻译。<br>
・“深海日志”图鉴中显示的角色现在可以更平滑地移动了。<br>
・使用“ヲ级的指挥权杖”选择深海栖舰, 舰娘现在变得更加迅速。<br>
・更改了海上的阵型移动逻辑。<br>
・提督现在可以吃下“零式21型舰战（1/48）”了。<br>
・提督现在可以喝下“修复桶”了。<br>
・稍微提高了与Morph MOD的兼容性。<br><br>

# 关于Shincolle 1.21.1α

这是从Shincolle 1.10.2非官方移植的MOD。包含大量未实现的功能和Bug。<br>
・目前是创造模式专用的MOD。<br>
・深海栖舰, 舰娘会跟随提督，并可以根据“ヲ级的指挥权杖”的指示进行移动和攻击。<br>
・除此以外的其他功能目前均未实现。<br>
