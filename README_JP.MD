# Shincolle 1.12.2βについて
Shincolle 1.10.2からの非公式移植MODです。元MODの一部改変を含みます。<br><br>
## 変更点一覧
・「超深海ノ熱水噴出孔」の渦巻きがよりスムーズに回転するようになりました。<br>
・深海棲艦、艦娘は、提督に積極的についてくるようになりました。<br>
・深海棲艦、艦娘は、トラップドアを開けられるようになりました。<br>
・深海棲艦、艦娘は、はしご使って移動しようと頑張るようになりました。<br>
・深海棲艦、艦娘のアイテム欄テクスチャに修正が加えられました。<br>
・深海棲艦、艦娘を右クリックしたとき、提督が手に持っているアイテムを使用してしまう問題が解消されました。<br>
・新たな設定項目「Consumption Mode」が追加されました。設定によって、建造に消費する燃料の量やチェストから出現するShincolle関連アイテムの数が変化します。<br>
・「深海提督ノ机」を横に並べると、テクスチャが変化するようになりました。<br>
・「深海提督ノ机」に新たな機能が追加されました。選択された深海棲艦、艦娘が机の前に集結します。<br>
・「深海提督ノ机」に関する重大な不具合が修正されました。<br>
・新しいブロック「深海提督ノ椅子」が追加されました。※作者はデザイナーではありません。<br>
・新しいブロック「ネザーノ深海金属鉱石」が追加されました。ネザーに生成されます。※作者はデザイナーではありません。<br>
・「ネザーノ深海金属鉱石」と「ポリメタル鉱石」にパーティクルが表示されるようになりました。<br>
・PVPモードの仕様が変更されました。設定がオフの時、提督が他の提督の深海棲艦、艦娘を攻撃しても反撃を行わなくなります。（きっと）<br>
・内部プログラムの最適化と画像の圧縮、一部サウンドの削除を行いました。それにより、ファイル容量が1MBほど削減されました。<br>
・「暁」に新しい装備が追加されました。<br>
・「深海提督ノ航海日誌(説明書)」に日本語翻訳が追加されました。
・「深海提督ノ航海日誌(説明書)」の図鑑に表示されるキャラクターがよりスムーズに動かせるようになりました。<br>
・「ヲ級ノ指揮ノ杖」を使った深海棲艦、艦娘の選択がより素早く行われるようになりました。<br>
・海上の陣形移動ロジックが変更されました。<br>
・提督は「零式艦戦21型 (1/48スケール)」を食べられるようになりました。<br>
・提督は「高速修復材」を飲めるようになりました。<br>
・Morph MODとの互換性が少し高まりました。<br><br>

# Shincolle 1.21.1αについて
Shincolle 1.10.2からの非公式移植MODです。未実装機能や不具合を多数含みます。<br>
・現在はクリエイティブモード専用MODです。
・深海棲艦、艦娘は提督を追従する、「ヲ級ノ指揮ノ杖」の指示で移動、攻撃をすることができます。
・それ以外の機能は現時点で未実装です。
